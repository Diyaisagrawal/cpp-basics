# cpp-basics

This repository contains concise examples demonstrating core C++ concepts,
including memory management, pointers, and usage of the STL. The goal is to
provide clear, well-documented examples suitable for learning and reference.

## Topics Covered
- Pointers and references
- Dynamic memory management
- Standard Template Library (STL)

## Environment
- C++
- Linux

Each example is intentionally small and focused on one concept.
